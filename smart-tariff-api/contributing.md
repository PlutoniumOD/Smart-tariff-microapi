
# Contributing to Smart Tariff Micro‑API

Thanks for your interest in improving the project!

## Reporting Issues
Open an issue on GitHub with:
- Clear description of the problem
- Add-on logs (if relevant)
- Steps to reproduce
- HA version, MQTT broker version, add‑on version

## Pull Requests
1. Fork the repository
2. Create a feature branch:
   `git checkout -b feature/my-change`
3. Commit changes with clear messages
4. Submit a PR

## Code Style
- Python 3.12+
- PEP8 formatting
- Prefer small focused commits
- Avoid UI-breaking config changes unless documented

## Testing
Before submitting a PR:
- Validate add-on builds cleanly
- Check logs for errors
- Confirm MQTT sensors appear correctly

## Contact
Open a discussion in GitHub if unsure where something fits.
